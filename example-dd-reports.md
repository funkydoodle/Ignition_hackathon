## SM Flow Line (in Channel Alert)

When Nansen smart money netflow data is available and meets thresholds, an SM Flow line is appended to the channel alert:

```
🟢 SM Inflow (1h): +$12.4K | 3 SM traders
```

For outflow:
```
🔴 SM Outflow (1h): -$8.2K | 2 SM traders
```

For tokens older than 2 hours, a 24h figure is also included:
```
🟢 SM Inflow (1h): +$5.1K (24h: +$42.3K) | 5 SM traders
```

## CTA Format

The call-to-action line uses Telegram HTML formatting:

```html
🔥 Trade on <a href="https://nocturnal.xyz/meme/{pool_address}">Nocturnal</a>
```

This renders as: `🔥 Trade on Nocturnal` where "Nocturnal" is a clickable hyperlink.

## Risk Factor Reference

### Factors that increase risk rating:

| Factor | Impact |
|--------|--------|
| Top 10 concentration > 60% | -> RED |
| Holders < 200 | -> RED |
| Liquidity < $10K | -> RED |
| Dev wallet > 10% | -> RED |
| No Twitter presence | +1 risk level |
| Sniper count > 10 | +1 risk level |
| Bearish buy/sell ratio | +1 risk level |

### Factors that decrease risk rating:

| Factor | Impact |
|--------|--------|
| Top 10 concentration < 40% | -> GREEN |
| Holders > 500 | -> GREEN |
| Liquidity > $25K | -> GREEN |
| Dexscreener paid | -1 risk level |
| Active GitHub (AI tokens) | -1 risk level |
| Strong narrative match | -1 risk level |
| High smart wallet count (7+) | -1 risk level |
